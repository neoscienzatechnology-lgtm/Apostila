Aula 2.1: Fundamentos do CSS

Módulo 2 - CSS | Duração: 2 horas

Objetivos da Aula

  • Entender sintaxe do CSS e formas de inclusão
  • Aprender seletores básicos e pseudo-classes
  • Compreender o box model e box-sizing
  • Praticar variáveis CSS e boa organização de estilos

1. Como adicionar CSS

Inline, internal e external — prefira arquivos externos para projetos reais.

Exemplo

<!-- index.html -->
<link rel="stylesheet" href="styles.css">

<!-- styles.css -->
:root{--primaria:#06b6d4}
.titulo{color:var(--primaria);font-size:2rem}

2. Seletores e especificidade

Entender especificidade evita conflitos de estilos.

/* ordem crescente de especificidade */
element { }
.classe { }
#id { }

/* uso prático */
.btn{background:#06b6d4;color:#fff;padding:.5rem 1rem}
.btn.primary{background:#0891b2}
.btn:hover{opacity:.95}

Dica: mantenha classes focadas em comportamento/semântica (ex.: .btn, .card) e evite estilizar por tags quando possível.

3. Box model e box-sizing

*{box-sizing:border-box}
.card{width:300px;padding:16px;border:1px solid #e5e7eb;margin:12px}
.container{max-width:1100px;margin:0 auto;padding:0 16px}

Recomendação: aplique box-sizing: border-box globalmente para evitar surpresas com largura e padding.

4. Variáveis CSS e temas

:root{
  --cor-bg:#ffffff;
  --cor-primaria:#06b6d4;
  --espaco:16px;
}

.site{background:var(--cor-bg);padding:var(--espaco)}

Usar variáveis facilita criar temas e ajustar valores globais rapidamente.

Exercícios sugeridos

  1. Crie um styles.css com variáveis e aplique em uma mini-página.
  2. Implemente uma .btn com estado :hover e :focus acessível.
