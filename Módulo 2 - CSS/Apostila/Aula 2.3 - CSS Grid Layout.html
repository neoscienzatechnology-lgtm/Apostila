Aula 2.3: CSS Grid Layout

Módulo 2 - CSS | Duração: 2 horas

Objetivos da Aula

  • Entender o modelo de layout bidimensional do Grid
  • Dominar o posicionamento preciso de elementos
  • Criar layouts complexos e responsivos
  • Combinar Grid com outras técnicas de layout

1. Fundamentos do Grid

CSS Grid é um sistema de layout bidimensional que permite controle preciso sobre linhas e colunas.

Conceitos Básicos

  • Grid Container: Elemento que recebe display: grid
  • Grid Items: Filhos diretos do container
  • Grid Lines: Linhas que formam a estrutura do grid
  • Grid Track: Espaço entre duas linhas do grid
  • Grid Cell: Intersecção entre uma linha e uma coluna
  • Grid Area: Retângulo formado por várias células
.container {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr;  /* 3 colunas */
    grid-template-rows: auto 1fr auto;    /* 3 linhas */
    gap: 1rem;                            /* espaçamento */
    min-height: 100vh;                    /* altura total */
}

2. Grid Template

Definindo Colunas e Linhas

.container {
    /* Unidades fixas */
    grid-template-columns: 200px auto 200px;

    /* Frações flexíveis */
    grid-template-columns: 1fr 2fr 1fr;

    /* Função repeat */
    grid-template-columns: repeat(3, 1fr);
    
    /* Auto-fill e auto-fit */
    grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
    
    /* Linhas automáticas */
    grid-auto-rows: minmax(100px, auto);
}

Áreas do Grid

.container {
    grid-template-areas:
        "header header header"
        "sidebar content content"
        "footer footer footer";
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }
.footer { grid-area: footer; }

3. Posicionamento e Alinhamento

Posicionamento de Items

.item {
    /* Por linha/coluna */
    grid-column: 1 / 3;        /* Início / Fim */
    grid-row: 2 / 4;          /* Início / Fim */
    
    /* Span (extensão) */
    grid-column: span 2;      /* Ocupa 2 colunas */
    grid-row: span 3;        /* Ocupa 3 linhas */
    
    /* Shorthand */
    grid-area: 2 / 1 / 4 / 3; /* row-start/col-start/row-end/col-end */
}

Alinhamento

.container {
    /* Alinhamento do Grid */
    justify-content: center;     /* Eixo inline (horizontal) */
    align-content: center;       /* Eixo block (vertical) */
    
    /* Alinhamento dos Items */
    justify-items: center;       /* Eixo inline (horizontal) */
    align-items: center;         /* Eixo block (vertical) */
}

.item {
    /* Alinhamento individual */
    justify-self: end;
    align-self: center;
}

4. Layouts Responsivos

Grid Responsivo Automático

.container {
    /* Grid responsivo que se ajusta automaticamente */
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 1rem;
}

/* Grid com diferentes layouts por breakpoint */
@media (max-width: 768px) {
    .container {
        grid-template-areas:
            "header"
            "content"
            "sidebar"
            "footer";
    }
}

Layout de Blog

.blog-layout {
    display: grid;
    gap: 2rem;
    grid-template-columns: repeat(3, 1fr);
    grid-template-areas:
        "featured featured featured"
        "post-1    post-2    post-3"
        "sidebar   sidebar   sidebar";
}

@media (min-width: 768px) {
    .blog-layout {
        grid-template-areas:
            "featured featured sidebar"
            "post-1   post-2   sidebar"
            "post-3   .        sidebar";
    }
}

5. Dicas Avançadas

Performance

  • Prefira grid-template-areas para layouts complexos
  • Use auto-fill/auto-fit com cuidado em grids grandes
  • Evite cálculos dinâmicos de posição em animações

Compatibilidade

  • Grid é bem suportado em navegadores modernos
  • Considere fallbacks para navegadores antigos
  • Teste subgrids em navegadores específicos

Boas Práticas

  • Use Grid para layouts macro, Flexbox para micro
  • Nomeie áreas do grid de forma semântica
  • Mantenha grids simples e focados
  • Documente layouts complexos com comentários
