Exercício 2.2: Header com Flexbox

Módulo 2 - CSS

Descrição

Desenvolva um header responsivo completo utilizando Flexbox que inclua:

  • Logo à esquerda
  • Menu de navegação à direita com 4-5 itens
  • Botão de ação (ex: "Contato" ou "Login")
  • Menu móvel que se adapta em telas menores

Requisitos

  • Use apenas Flexbox para o layout (não use Grid)
  • O header deve ter altura fixa e background com cor ou gradiente
  • Os itens do menu devem ter hover state
  • O botão de ação deve ter estilo diferenciado
  • Em mobile (< 768px):
    • Menu deve ficar abaixo do logo
    • Itens do menu devem se organizar verticalmente
    • Botão deve ocupar largura total

Desafios Extras

  • Adicione um submenu dropdown em um dos itens do menu
  • Implemente um menu burger para mobile com animação
  • Adicione uma transição suave ao alternar entre desktop/mobile
  • Faça o header mudar de cor ao rolar a página

Dicas

  • Use unidades relativas (rem, em) para espaçamentos
  • Aproveite a propriedade gap para espaçamento uniforme
  • Considere usar z-index para dropdowns
  • Teste em diferentes tamanhos de tela
  • Garanta que o layout não quebre em resoluções intermediárias

Estrutura Base

Você pode começar com esta estrutura HTML:

<header class="header">
    <div class="header-container">
        <div class="logo">
            <a href="/">Logo</a>
        </div>
        
        <nav class="nav-menu">
            <ul class="nav-list">
                <li><a href="#">Home</a></li>
                <li><a href="#">Sobre</a></li>
                <li><a href="#">Serviços</a></li>
                <li><a href="#">Blog</a></li>
            </ul>
            
            <button class="cta-button">
                Contato
            </button>
        </nav>
    </div>
</header>
