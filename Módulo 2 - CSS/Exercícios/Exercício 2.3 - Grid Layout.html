Exercício 2.3: Layout de Blog com Grid

Módulo 2 - CSS

Descrição

Desenvolva um layout completo de blog usando CSS Grid que inclua:

  • Header com logo e navegação
  • Hero section destacada
  • Grid de posts principais
  • Sidebar com widgets
  • Footer com múltiplas colunas

Requisitos

  • Use grid-template-areas para definir o layout macro
  • O grid de posts deve ser responsivo e automático
  • A sidebar deve ter largura fixa em desktop
  • O hero deve ocupar largura total
  • Em mobile (< 768px):
    • Sidebar deve ir para baixo do conteúdo
    • Posts devem ocupar largura total
    • Footer deve reorganizar as colunas
  • Implement gap consistente entre elementos

Desafios Extras

  • Adicione um layout especial para posts destacados
  • Implemente um grid de imagens na galeria com aspect-ratio
  • Crie um layout alternativo para tablets (768px - 1024px)
  • Adicione animações nas transições de layout

Dicas

  • Use nomes semânticos para as áreas do grid
  • Aproveite minmax() para células flexíveis
  • Considere usar auto-fit para grids responsivos
  • Teste diferentes resoluções e conteúdos
  • Use grid dev tools do navegador para debug

Estrutura Base

Comece com esta estrutura HTML:

<div class="blog-layout">
    <header class="header">
        <nav class="nav">
            <!-- Navegação aqui -->
        </nav>
    </header>

    <section class="hero">
        <!-- Hero content -->
    </section>

    <main class="posts-grid">
        <article class="post">
            <!-- Post content -->
        </article>
        <!-- Mais posts... -->
    </main>

    <aside class="sidebar">
        <div class="widget">
            <!-- Widget content -->
        </div>
        <!-- Mais widgets... -->
    </aside>

    <footer class="footer">
        <div class="footer-grid">
            <!-- Footer columns -->
        </div>
    </footer>
</div>
