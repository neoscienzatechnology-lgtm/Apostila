Exercício 2.5: CSS Frameworks

Neste exercício, você irá criar um dashboard moderno utilizando Bootstrap e Tailwind CSS.

Instruções
  1. Implemente um dashboard com:
    • Navbar responsiva com menu hamburguer em mobile
    • Sidebar com navegação vertical
    • Grid de cards com estatísticas
    • Gráficos (pode usar Chart.js)
    • Tabela de dados responsiva
    • Formulário de configurações
  2. Use Bootstrap para o layout base e componentes
  3. Estilize detalhes com Tailwind para melhor aparência
  4. Implemente tema claro/escuro
  5. Garanta responsividade em todos os breakpoints
