Projeto Final: Landing Page Moderna

Módulo 2 - CSS

Descrição do Projeto

Desenvolva uma landing page moderna e responsiva para um produto ou serviço de sua escolha. O objetivo é aplicar todos os conceitos aprendidos no módulo de CSS, incluindo layouts responsivos, animações, e frameworks modernos.

Requisitos Técnicos

1. Layout e Responsividade

  • Design responsivo para mobile, tablet e desktop
  • Grid system consistente
  • Navigation bar responsiva com menu mobile
  • Hero section com imagem e call-to-action
  • Footer com múltiplas colunas

2. Componentes

  • Cards de features/produtos
  • Formulário de contato/newsletter
  • Seção de depoimentos
  • Galeria de imagens
  • Modal ou popups

3. Estilização

  • Tema de cores consistente
  • Tipografia hierárquica clara
  • Animações e transições suaves
  • Ícones e elementos visuais
  • Micro-interações

4. Performance e Organização

  • Código CSS organizado e documentado
  • Otimização de imagens e recursos
  • Tempo de carregamento otimizado
  • Integração correta de frameworks
