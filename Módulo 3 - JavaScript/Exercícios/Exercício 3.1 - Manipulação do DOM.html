Exercício 3.1 - Soma de Arrays

Objetivo: implementar uma função pura que recebe um array de números e retorna um objeto com:

  • total (soma)
  • média
  • mínimo
  • máximo

Requisitos:

  1. Use métodos funcionais (reduce, map quando necessário).
  2. Trate arrays vazios (retorne valores coerentes).
  3. Não modifique o array original.

Critérios de avaliação: correção, legibilidade, tratamento de casos extremos e uso apropriado de métodos funcionais.
