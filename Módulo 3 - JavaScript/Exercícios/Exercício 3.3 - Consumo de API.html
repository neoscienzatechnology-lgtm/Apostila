Exercício 3.3 - DOM Manipulation

Objetivo: implementar interações dinâmicas no DOM usando boas práticas (delegação, DocumentFragment e templates).

Tarefas

  1. Crie uma interface que renderize uma lista a partir de um array de objetos {id, nome}.
  2. Implemente um campo para adicionar novos itens à lista (sem recarregar a página).
  3. Implemente um botão de remover ao lado de cada item. Use delegação de eventos para gerenciar os cliques.
  4. Use <template> para o markup de cada item e DocumentFragment ao inserir múltiplos itens.
  5. Garanta que o foco seja gerenciado corretamente ao adicionar/remover itens (usabilidade).

Requisitos

  • Não utilize frameworks; apenas DOM API nativa.
  • Não gere vazamentos de event listeners ao remover elementos.
  • Trate entradas vazias e IDs duplicados.

Critérios: funcionamento correto, performance (uso de fragmentos), código limpo e acessibilidade básica (atributos ARIA e gerenciamento de foco).
