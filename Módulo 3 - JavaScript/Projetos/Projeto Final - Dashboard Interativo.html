Projeto Final - Dashboard Interativo

Descrição do Projeto

Neste projeto final, você irá desenvolver um Dashboard Interativo que integra todos os conceitos aprendidos durante o módulo de JavaScript. O dashboard permitirá visualizar e gerenciar dados através de diferentes componentes interativos.

Requisitos Funcionais

  • Implementar um sistema de autenticação simples (login/logout)
  • Criar um gráfico de barras ou linha usando uma biblioteca de sua escolha (Chart.js recomendado)
  • Desenvolver uma lista de tarefas (Todo List) com CRUD completo
  • Implementar um sistema de filtros e busca para os dados
  • Criar um formulário para adicionar novos dados ao dashboard
  • Implementar persistência de dados usando localStorage
  • Adicionar pelo menos uma chamada a uma API externa (sugestão: JSONPlaceholder)

Requisitos Técnicos

  • Utilizar HTML5 semântico para estruturar a página
  • Implementar design responsivo usando CSS (Tailwind CSS recomendado)
  • Usar JavaScript moderno (ES6+)
  • Implementar tratamento de erros adequado
  • Utilizar Promises/Async/Await para chamadas assíncronas
  • Implementar pelo menos um Event Delegate
  • Usar localStorage para persistência de dados
  • Documentar o código adequadamente
