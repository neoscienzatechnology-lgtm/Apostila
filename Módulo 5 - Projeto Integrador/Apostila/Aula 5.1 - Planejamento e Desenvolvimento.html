Aula 5.1: Planejamento e Desenvolvimento

Módulo 5 - Projeto Integrador | Duração: 2 horas

Objetivos

  • Planejar um projeto integrador usando as tecnologias estudadas
  • Definir requisitos, arquitetura e divisão de tarefas
  • Preparar para deploy e apresentação final

1. Fases do projeto

  1. Definição de requisitos e escopo
  2. Prototipação (wireframes)
  3. Implementação incremental
  4. Testes e ajustes
  5. Deploy e apresentação

Checklist rápido

  • Requisitos mínimos definidos
  • Arquitetura (frontend, backend, banco) esboçada
  • Rotina de deploy documentada

Exercício

Escreva um plano curto (máx. 1 página) para um projeto pessoal que integre HTML, CSS, JavaScript e banco de dados. Liste funcionalidades principais, telas e dados necessários.
